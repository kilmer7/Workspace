1º abri o bash no diretorio do projeto, se vc estiver no diretorio certo vai aparecer a branch que vc esta trabalhando no final entre parente, no caso agora vc ta na master

2º execute git checkout -b "nome da nova branch" , para mudar para uma nova branch

3º git add . para subir todas as alterações feitas

4º git commit , ira abri a tela no bash para escrever o commit, aperte "insert" para poder escrever, despois de escrever bunitinho, aperta "esc" e  depois digite :wq para  sair e salvar o commit

5º git push origin "nome da branch que vc quer subir"

6º dentro do git hub tu mergea oque vc acabou de subir com a master

7º git checkout oringin master

8º git pull oringin  master , para atualizar a master local
